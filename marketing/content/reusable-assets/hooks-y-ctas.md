# Hooks y CTAs reutilizables

Banco vivo. Al medir contenido, mover los ganadores arriba y anotar su performance.

## Hooks probados internamente (origen: scripts y copy del sitio)

- "La mayoría de los negocios no necesita IA."
- "Tu WhatsApp vende. Y también pierde ventas todos los días."
- "Tu web no es tu problema. Tu seguimiento sí."
- "El que preguntó el martes, el jueves ya compró en otro lado."
- "Deja de perseguir curiosos."
- "¿Cuánto vale para ti cerrar 1 cliente extra este mes?"
- "Precio, horario, ubicación. Las mismas 6 preguntas, 40 veces por semana."

## Estructuras de hook

- Contraintuitivo: "No necesitas [lo que crees]. Necesitas [lo real]."
- Patrón observado: "De N [negocios] que revisé esta semana, X tienen [problema]."
- Costo invisible: "Cada [pregunta sin responder] te cuesta [consecuencia concreta]."
- Detrás de cámara: "Así funciona [sistema interno real] que construí para [resultado]."

## CTAs (uno por pieza, nunca dos)

| CTA | Cuándo usarlo |
|---|---|
| "Escríbeme 'diagnóstico'" | Piezas P1/P3 con dolor activado |
| Link a `/cotizar` | Piezas P4 de oferta directa |
| Link a página vertical (`/clinicas`, etc.) | Piezas dirigidas a un vertical |
| "Guarda este post" | Piezas P3 de utilidad (alimenta alcance) |
| "Sígueme para ver cómo se construye" | Piezas P2 build in public |

## Frases de oferta (consistentes con el sitio — no alterar precios)

- "Diagnóstico express: USD 150, entrega en 48h, acreditable si construimos después."
- "Landing page: USD 199, lista en 3 días."
- "Automatización desde USD 499. Web o producto desde USD 699."
- "Precios publicados, entrega en días, sin cotización a ciegas."
