# published/

Registro permanente de contenido publicado. Cada archivo llega desde `approved/` con estos campos añadidos por el Publisher Agent tras verificar la publicación:

```
url_publicada: <link directo>
fecha_hora: YYYY-MM-DD HH:MM (zona horaria)
plataforma: instagram | linkedin | x | blog | newsletter
evidencia: <screenshot en esta carpeta o descripción verificable>
```

A los 7 días de publicada, el Analytics Agent registra su performance en `analytics/content-performance.csv`.
