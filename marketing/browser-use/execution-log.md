# Log de ejecución — Browser Use

Toda corrida deja entrada, exitosa o no. Acción sin registro = acción no realizada. Más reciente arriba.

## Formato

```markdown
### [YYYY-MM-DD HH:MM TZ] WF-nn — plataforma
- **Operador:** agente (sesión) | humano asistido
- **Objetivo:** [qué se buscaba]
- **Resultado:** éxito | parcial | fallo | detenido-escalado
- **Evidencia:** [URLs, archivos, screenshots]
- **Errores/observaciones:** [detalle; si se detuvo, por qué]
```
