# Plantilla de brief (copiar para cada pieza)

```markdown
# Brief — [título de trabajo]

- **Objetivo comercial:** [qué debe producir: DMs, visitas a /cotizar, autoridad para outreach de X vertical]
- **Audiencia:** [vertical + rol; ¿en qué etapa del funnel está?]
- **Mensaje principal:** [1 frase; qué debe recordar el lector]
- **Hook:** [primera línea/slide; sostiene solo]
- **Contenido:** [argumento, estructura, datos disponibles CON fuente]
- **CTA:** [una sola acción]
- **Formato:** [post / carrusel N slides / hilo / guion video / guía]
- **Canal:** [instagram / linkedin / x / blog / newsletter]
- **Fecha propuesta:** [YYYY-MM-DD]
- **Métrica esperada:** [número concreto: "2 DMs", "5 guardados"]
- **Hipótesis:** [por qué creemos que funcionará]
- **Estado:** idea | brief | borrador | revisión | aprobado | publicado
- **Reutilización prevista:** [en qué se convertirá después]
```

Regla: si "Contenido" necesita un dato que no existe, el brief vuelve con la observación — no se redacta con datos inventados.
